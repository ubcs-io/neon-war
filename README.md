# neon-war
A simple python game emulating the card game 'war', generated as a demo of GLM 4.7 capabilities.

<img width="1430" height="835" alt="neon-war" src="https://github.com/user-attachments/assets/5407b7a9-ba74-40d5-8af4-696a3c50db56" />

# Download & Run Locally
Save the main file, then install flask.  Navigate to the file and run it:
```
pip install flask
python card_war.py
```
Then play from your browser by visiting `http://127.0.0.1:5000/`
